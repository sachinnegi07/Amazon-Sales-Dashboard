# Amazon Sales Dashboard

## 📌 Project Overview

This project presents an interactive Excel dashboard created to analyze Amazon sales data.  
The dashboard helps to understand revenue, profit, sales performance, and product trends in a clear visual format.

---

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Visualization
